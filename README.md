# Pi LED Game
This game gives the player a generated pattern, that the player has to repeat. Every next level the pattern increases by one led.

## PIN assignment
Button 1 : GPIO 21 </br>
Button 2 : GPIO 20 </br>
Button 3 : GPIO 16 </br>
Button 4 : GPIO 26

LED 1 : GPIO 5 </br>
LED 2 : GPIO 6 </br>
LED 3 : GPIO 13 </br>
LED 4 : GPIO 19
